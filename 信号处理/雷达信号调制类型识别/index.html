    雷达信号调制类型识别¶

    本章整理自《基于FPGA多门限时域的数字雷达信号动态检测与调制识别研究》第四章

    4.1 脉内调制识别方法¶

    4.1.1 基于瞬时频率的调制识别方法¶

    核心结论¶

    不同调制方式会产生不同的瞬时频率变化规律，因此可通过提取瞬时频率完成调制识别。

    关键公式¶

    瞬时频率：

    \[ f(n)=\frac{1}{2\pi}[\varphi(n)-\varphi(n-1)]f_s \tag{4-1} \]

    瞬时自相关：

    \[ R(n,m)=s^*(n)s(n+m) \tag{4-2} \]

    瞬时相位：

    \[ \theta(n,m)=\arctan\left[\frac{\operatorname{Im}(R_s(n,m))}{\operatorname{Re}(R_s(n,m))}\right] \tag{4-3} \]

    改写后的瞬时频率：

    \[ f(n,m)=\frac{\theta(n,m)f_s}{2\pi} \tag{4-4} \]

    符号说明¶

    • \(f(n)\)：瞬时频率
    • \(\varphi(n)\)：第 \(n\) 个采样点的相位
    • \(f_s\)：采样频率
    • \(R(n,m)\)：瞬时自相关结果
    • \(s(n)\)：输入信号
    • \(\theta(n,m)\)：瞬时相位
    • \(\operatorname{Re}(\cdot)\)：实部
    • \(\operatorname{Im}(\cdot)\)：虚部

    粗分类思路¶

    通过功率谱 3 dB 带宽先把信号分为两类： - FM：频谱较宽 - PSK：频谱较窄

    方法不足¶

    单独依靠瞬时频率时，对频率估计精度依赖较强，低信噪比下识别鲁棒性不足。

    4.1.2 基于瞬时相关和瞬时频率的调制识别方法¶

    核心结论¶

    本文最终采用组合方法： - 前端用 3 dB 带宽进行 FM/PSK 粗分类 - PSK 用瞬时相关和峰值种类数识别 - FM 用八点滑动均值测频，再按频率变化趋势识别

    PSK 信号识别¶

    核心原理¶

    对于调相信号： - NS：无峰值 - BPSK：1 类峰值 - QPSK：2 类峰值

    为了让峰值在低信噪比下更明显，文中采用汉宁窗滤波对相关输出做幅值修正。

    关键公式¶

    汉宁窗：

    \[ \omega(n)=0.5\left[1-\cos\left(\frac{2\pi n}{M+1}\right)\right],\quad 1\le n\le M \tag{4-5} \]

    符号说明¶

    • \(\omega(n)\)：窗函数系数
    • \(n\)：窗内序号
    • \(M\)：窗长

    工程结论¶

    • 加窗后噪底更平滑
    • BPSK 与 QPSK 峰值更突出
    • QPSK 第二类峰值更容易从噪声中分离出来

    FM 信号识别¶

    核心原理¶

    通过相位差分得到瞬时频率，再进行： - 相位解卷绕 - 相位翻转修正 - 八点滑动均值降噪

    最终根据频率变化趋势区分： - LFM：线性变化 - FSK：跳变频率 - NLFM：非线性变化

    工程结论¶

    • 八点滑动均值的测频精度优于直接法和四点平均法
    • 在 SNR = 5 dB 时，频率误差可控制在约 0.26% 以内
    • 随着信噪比提高，频率误差进一步降低

    4.2 信号调制识别的 FPGA 设计¶

    总体结构¶

    调制识别流程分两部分： 1. 3 dB 频谱带宽粗识别 2. PSK 与 FM 两条支路分别细分类识别

    配图记录¶

    4.2.1 3dB 频谱带宽模块设计¶

    核心结论¶

    对输入 IQ 数据进行： - 2048 点截取 - 加窗 - FFT - CORDIC 求模 - log 运算 - 3 dB 带宽统计

    再将结果与阈值比较，完成 FM / PSK 粗分类。

    工程点¶

    • 基于 FFT IP core 实现
    • 接口遵循 AXI 握手协议
    • 以阈值完成粗分类

    典型结果¶

    • LFM 的 3 dB 带宽较宽
    • NS 的 3 dB 带宽很窄
    • 因而可以完成可靠粗分类

    第四章完成了雷达脉内调制识别方案的理论、仿真与硬件实现： - 用 3 dB 带宽完成 FM / PSK 粗分类 - 用瞬时相关识别 NS、BPSK、QPSK - 用改进测频识别 LFM、FSK、NLFM - 最终实现 6 类信号的有效识别，并验证了 FPGA 实现可行性
