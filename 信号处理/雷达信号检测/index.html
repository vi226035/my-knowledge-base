    雷达信号检测¶

    本章整理自《基于FPGA多门限时域的数字雷达信号动态检测与调制识别研究》第三章

    3.1 常见的信号检测方法分析¶

    3.1.1 滑动能量积累检测法¶

    核心结论¶

    通过滑动窗对能量累加，并与门限比较来判断信号是否存在。思路简单，但低信噪比下脉冲包络波动大，门限设置困难。

    3.1.2 自相关检测法¶

    核心结论¶

    利用信号的相关性和噪声的不相关性，通过自相关运算增强真脉冲、抑制噪声，是更适合 FPGA 实时实现的检测方法。

    关键公式¶

    输入信号：

    \[ X(n)=Ae^{j\varphi}e^{j2\pi fn\Delta t}+\omega(n) \tag{3-1} \]

    自相关输出：

    \[ R(n)=\frac{1}{N}\sum_{i=n-N-1}^{n-2}X(i+1)X^*(i) \tag{3-2} \]

    相关后信噪比：

    \[ SNR_R=\frac{NA^4}{(2A^2+\sigma^2)\sigma^2}=\frac{NA^2}{2A^2+\sigma^2}SNR_X \tag{3-4} \]

    递推实现：

    \[ R(n)=R(n-1)+\frac{1}{N}[X(n-1)X^*(n-2)-X(n-N-1)X^*(n-N-2)] \tag{3-5} \]

    符号说明¶

    • \(X(n)\)：输入复包络信号
    • \(A\)：信号幅度
    • \(\varphi\)：初始相位
    • \(f\)：载频或瞬时频率分量
    • \(\Delta t\)：采样间隔
    • \(\omega(n)\)：噪声项
    • \(R(n)\)：自相关输出
    • \(N\)：平均长度
    • \(SNR_X\)：输入信噪比
    • \(SNR_R\)：相关后信噪比
    • \(\sigma^2\)：噪声方差

    工程结论¶

    • 自相关后脉冲更突出，门限判决更容易。
    • 递推形式便于流水线硬件实现。
    • 在检测方法比较后，本文最终选用自相关法。

    3.1.3 小波阈值去噪检测法¶

    核心结论¶

    通过多层小波分解和阈值处理实现去噪，再重构信号。去噪效果较好，但流程复杂，不利于高实时性硬件实现。

    关键公式¶

    \[ f=A+D3+D2+D1 \tag{3-6} \]

    符号说明¶

    • \(f\)：原始信号
    • \(A\)：低频近似分量
    • \(D1,D2,D3\)：各层高频细节分量

    3.2 脉冲门限检测方法分析¶

    3.2.1 噪声方差法¶

    核心结论¶

    通过先估计噪声方差，再由噪声统计分布设定检测门限。方法理论清晰，但门限依赖信噪比，工程灵活性较差。

    关键公式¶

    噪声标准差估计：

    \[ \hat{\sigma}=\sqrt{\frac{\sum_{i=1}^{n}n_i^2}{N-1}} \tag{3-7} \]

    噪声模值分布参数：

    \[ \mu=1.235\sqrt{2N}\sigma^2 \tag{3-8} \]
    \[ \sigma_1=0.43\sqrt{2N}\sigma^2 \tag{3-9} \]

    检测门限：

    \[ V_T=\mu+a\sigma_1 \tag{3-10} \]

    符号说明¶

    • \(\hat{\sigma}\)：噪声标准差估计值
    • \(n_i\)：第 \(i\) 个噪声样本
    • \(N\)：样本数
    • \(\mu\)：噪声模值均值参数
    • \(\sigma_1\)：噪声模值标准差参数
    • \(a\)：门限调整系数
    • \(V_T\)：检测门限

    3.2.2 相关差值法¶

    核心结论¶

    这是本文脉冲检测的核心改进方法。利用相关后包络存在前后沿的特点，对包络做隔点差分，统计极值，构造高低双门限： - 高门限判断脉冲到达 - 低门限判断脉冲结束

    关键公式¶

    类内方差：

    \[ \sigma_w^2(V_T)=\left[\frac{N_1(V_T)}{N(V_T)}\right]^\beta \sigma_1^2(V_T)+\left[\frac{N_2(V_T)}{N(V_T)}\sigma_2^2(V_T)\right] \tag{3-11} \]

    类间方差：

    \[ \sigma_b^2(V_T)=\left[\frac{N_1(V_T)}{N(V_T)}\right]^\beta |M_r(V_T)-M_1(V_T)|+\left[\frac{N_2(V_T)}{N(V_T)}\right]|M_r(V_T)-M_2(V_T)| \tag{3-12} \]

    门限统计量：

    \[ V(\beta,V_T)=\beta\sigma_b^2(V_T)/\sigma_w^2(V_T) \tag{3-13} \]

    符号说明¶

    • \(\sigma_w^2(V_T)\)：类内方差
    • \(\sigma_b^2(V_T)\)：类间方差
    • \(V_T\)：门限值
    • \(N_1(V_T)\)：第一类样本数
    • \(N_2(V_T)\)：第二类样本数
    • \(N(V_T)\)：总样本数
    • \(M_1(V_T), M_2(V_T)\)：两类样本均值
    • \(M_r(V_T)\)：参考均值
    • \(\beta\)：加权指数
    • \(V(\beta,V_T)\)：门限统计量

    工程结论¶

    • 在 \(\beta=2\) 时，八点差值效果最好。
    • 方法虽不一定达到理论最优，但已足够接近，且实现简单。
    • 相比噪声方差法，虚警率更低，时域参数测量更准。

    结果结论¶

    • 3 dB 时虚警概率约接近 10%
    • 5 dB 时虚警概率接近 0
    • 到达时间和脉宽测量精度明显优于噪声方差法

    3.3 信号检测的 FPGA 实现¶

    3.3.1 自相关算法模块设计¶

    核心结论¶

    硬件实现分为三部分： 1. 复数共轭乘法 2. 延迟链 3. 三路加减递推累加

    关键公式¶

    \[ R(n)=R(n-1)+\frac{1}{N}[X(n-1)X^*(n-2)-X(n-N-1)X^*(n-N-2)] \tag{3-14} \]

    符号说明¶

    • \(R(n)\)：当前自相关输出
    • \(R(n-1)\)：上一时刻自相关输出
    • \(X(n)\)：输入复信号
    • \(N\)：平均长度

    工程实现点¶

    • 复数乘法器用于相邻采样点共轭相乘
    • 小规模延迟可用 SRLC32E、FDRE 实现，节省资源
    • 后续通过 CORDIC 求模得到包络数据

    3.3.2 脉冲检测模块设计¶

    核心结论¶

    整体流程： 1. 输入信号做自相关 2. 经 CORDIC 求模 3. 八点差值求高低门限 4. 延迟后的包络与门限比较 5. 输出脉冲标识位 FLAG 和脉宽等参数

    判决机制¶

    • 大于高门限时判定脉冲开始
    • 小于低门限时判定脉冲结束
    • 连续多点确认提高鲁棒性

    实测结论¶

    • 脉宽测量误差约 1% 左右
    • 漏警率接近 0
    • 硬件资源占用较低，便于工程部署
