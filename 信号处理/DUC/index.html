    数字上变频（DUC）¶

    一句话理解¶

    DUC（Digital Up Converter）把低速基带 I/Q 复信号搬移到中频/射频，同时提高采样率，生成适合 DAC 发射的高速实信号。

    本质就三步：内插升采样 → 低通滤波抗镜像 → 混频搬频。正好是 DDC 的逆过程。

    为什么需要 DUC？¶

    在雷达发射机、通信发射链路中：

    • 基带处理（调制、脉冲成型）在低采样率下完成，省资源
    • DAC 需要高采样率来减小零阶保持效应、拉开镜像距离
    • 最终发射的是中频/射频实信号（天线只能发射实信号）

    零阶保持效应：DAC 在两个采样点之间保持上一个值不变（像楼梯台阶），这会在频谱上引入高频衰减（sinc 滚降）。采样率越高，台阶越密，衰减越少。就像用更多像素来描述一条曲线——像素越多锯齿越不明显。

    DUC 负责桥接低速基带和高速 DAC——先升采样，再搬频，最后合成实信号。

    数学推导¶

    输入信号模型¶

    基带 I/Q 复信号（低采样率 \(f_{s,in}\)）：

    \[z[m] = I[m] + jQ[m] = A[m] \cdot e^{j\phi[m]}\]

    其中 \(A[m]\) 是幅度，\(\phi[m]\) 是相位/频率调制。目标采样率 \(f_{s,out}\)，内插因子 \(L = f_{s,out} / f_{s,in}\)。

    内插（Interpolation）¶

    在相邻采样点间插入 \(L-1\) 个零：

    \[z_{up}[n] = \begin{cases} z[m] & n = mL \\ 0 & \text{otherwise} \end{cases}\]

    插零后频谱是原始频谱的 \(L\) 倍周期延拓（因为采样率名义上提高了 \(L\) 倍，但实际有效样本没增加）。

    抗镜像滤波¶

    插零引入了镜像频谱（images），需要用低通滤波器（也叫内插滤波器）去除：

    \[z_{lp}[n] = h_{lp}[n] * z_{up}[n]\]

    通带截止频率：\(\omega_p = 2\pi \cdot B / f_{s,out}\)（基带信号带宽以内） 阻带起始频率：\(\omega_s = 2\pi \cdot f_{s,in} / (2 \cdot f_{s,out}) = \pi / L\)（镜像开始处）

    顺序很重要：先插零再滤波！这样才能去除插零引入的镜像频谱。实际上插零和滤波可合并为多相结构实现，滤波跑在低采样率一侧。

    镜像是什么？ 插零相当于强行提高了"名义采样率"，但新插入的零不包含真实信息，频谱上会在原信号的副本（镜像）出现在新采样率间隔的各个位置。低通滤波就是把这些多余的"影子"信号过滤掉，只保留基带部分的真实信号。

    正交上变频¶

    将滤波后的基带复信号与 NCO 混频，搬移到中频 \(\omega_c\)。

    I/Q 两路分别乘 NCO 的 \(\cos\) 和 \(-\sin\) 分量，最后相加：

    \[\begin{aligned} x[n] &= I[n] \cdot \cos(\omega_c n) + Q[n] \cdot (-\sin(\omega_c n)) \\ &= I[n]\cos(\omega_c n) - Q[n]\sin(\omega_c n) \end{aligned}\]

    注意：Q 路乘 \(-\sin\)（不是 \(+\sin\)），然后 I/Q 两路做加法。如果你看到的资料用的是 Q 路乘 \(+\sin\) 再减法，结果一样——只是约定不同。用加法更直观地对应了复乘的实部提取。

    用欧拉公式可验证，上式恰好等于 \(\text{Re}\{(I[n] + jQ[n]) \cdot e^{j\omega_c n}\}\)，即复基带信号被搬移到了 \(\omega_c\) 处。

    \[\begin{aligned} (I + jQ) \cdot (\cos\omega_c + j\sin\omega_c) &= \underbrace{(I\cos\omega_c - Q\sin\omega_c)}_{\text{实部}} + j(Q\cos\omega_c + I\sin\omega_c) \end{aligned}\]

    取实部：\(I[n]\cos(\omega_c n) - Q[n]\sin(\omega_c n)\)，正是 DAC 需要的实信号。

    输出信号¶

    \[x[n] = \text{Re}\left\{A[n] \cdot e^{j(\omega_c n + \phi[n])}\right\} = A[n] \cdot \cos(\omega_c n + \phi[n])\]

    成功！基带的幅度 \(A[n]\) 和相位 \(\phi[n]\) 原封不动地搬到了载波频率 \(\omega_c\) 上。

    完整信号流程¶

    I[m] ──→ [↑L] ──→ [LPF] ──→ [× cos(ω_c n)] ──┐
                                     ↓          ├──→ x[n] → DAC
Q[m] ──→ [↑L] ──→ [LPF] ──→ [× -sin(ω_c n)] ──┘
         ↑
    内插滤波器             ↑
                      NCO (数控振荡器)

    核心组成模块¶

    1. 内插器（Interpolator）¶

    内插因子 \(L\) 的选择依据：

    \[L = \frac{f_{s,out}}{f_{s,in}} = \frac{\text{DAC采样率}}{\text{基带采样率}}\]

    多相实现：插零 + 滤波可以用多相滤波器结构高效实现——滤波运算跑在低采样率一侧（\(f_{s,in}\)），只在高采样率侧输出：

    \[\begin{aligned} y[n] &= \sum_{k} h[k] \cdot x_{up}[n-k] \\ \text{第 }p\text{ 相输出}: y[mL + p] &= \sum_{r} h[rL + p] \cdot x[m-r] \end{aligned}\]

    \(p = 0, 1, ..., L-1\)，每相都是一个工作在 \(f_{s,in}\) 的子滤波器。

    2. 内插滤波器（LPF / Image-Rejection Filter）¶

    与 DDC 中的抽取滤波器功能对称，用于抑制插零产生的镜像。

    滤波器类型： - FIR：线性相位，稳定，最常用 - CIC 插值滤波器：无乘法器，适合大倍率内插的第一级 - 半带滤波器：约一半系数为 0，计算量减半

    多级内插（与 DDC 多级抽取对应）：

    输入 → [FIR] → [↑2, HB] → [↑2, HB] → [↑L, CIC] → 输出

    总内插倍数 \(L = L_1 \times 2 \times 2 \times L_4\)。顺序与 DDC 相反——先小倍数内插 + FIR 整形，最后大倍数 CIC。

    3. NCO（数控振荡器）¶

    与 DDC 中的 NCO 完全一致： - 相位累加器 + LUT 查找表 - 同时产生 \(\cos(\omega_c n)\) 和 \(\sin(\omega_c n)\)（或 \(-\sin\)，看约定） - 频率步进 \(\Delta\theta = 2\pi f_c / f_{s,out}\)

    4. 混频与合成¶

    I/Q 两路分别乘 NCO 输出后，再做减法合成：

    \[x[n] = I[n] \cdot \cos(\omega_c n) - Q[n] \cdot \sin(\omega_c n)\]

    注意这里是减号，因为调制公式源于复指数展开。用加号会导致频谱搬移方向相反。

    FPGA 实现要点¶

    位宽规划（与 DDC 对称）¶

    • 内插滤波后位宽：\(B_{out} = B_{in} + \lceil \log_2(\sum |h_k|) \rceil\)
    • CIC 内插后位宽：\(B_{out} = B_{in} + N \cdot \log_2(L)\)
    • 混频后截位送 DAC：需要做舍入（rounding），避免直流偏置

    多相内插滤波器¶

    这是 DUC 中最重要的优化技巧，让滤波器运算跑在低采样率一侧：

    传统做法：插零 → 在 f_s_out 时钟下滤波 → 大部分乘法乘的是 0，浪费
多相做法：L 个子滤波器并行工作 → 各跑在 f_s_in 时钟 → 输出再拼成 f_s_out

    FPGA 中用 \(L\) 个并行子滤波器 + 输出轮询，等效实现了高速内插。

    通俗理解：插零后的序列大部分是 0，传统 FIR 要对这些 0 也做乘加——白费力气。多相结构相当于先把滤波器系数按位置分组，每组只在有"真数据"的时钟周期做乘加，然后轮流输出结果。就像一个轮班制的厨师团队，每人只负责一道菜的上菜时刻，而不是所有人守在灶台前等每一次上菜。

    CIC 内插¶

    Hogenauer 结构的 CIC 内插器：先积分（在低速侧），再梳状差分（在高速侧）。与 CIC 抽取结构互为镜像。

    资源优化建议¶

    • DSP48 乘法器用于 FIR 和非 CIC 滤波
    • CIC 仅用加法器 + 寄存器和查找表实现，节省 DSP 资源
    • 利用 FIR 的系数对称性减少乘法器用量
    • 合理插入流水线寄存器平衡时序，满足 \(f_{s,out}\) 的时序需求

    DAC 接口¶

    DUC 输出直接驱动 DAC。需要注意： - 输出位宽匹配 DAC 分辨率 - 必要时加抖动（dither）改善小信号下的量化杂散 - 输出速率 = DAC 采样时钟频率

    关键指标¶

    指标 含义 典型值
    基带输入率 I/Q 采样率 几 MHz ~ 几十 MHz
    输出采样率 DAC 速率 100 MHz ~ 数 GHz
    内插倍数 升采样率比 10 ~ 10000
    输出频率 中频/射频 几十 ~ 几百 MHz
    SFDR 无杂散动态范围 > 70 dBc
    镜像抑制 对内插镜像的衰减 > 80 dB

    DDC vs DUC 对比¶

    特性 DDC（接收） DUC（发射）
    信号流向 ADC → 基带 基带 → DAC
    采样率变化 降低（抽取 ↓D） 提高（内插 ↑L）
    混频方向 向下搬移 向上搬移
    滤波目的 抗混叠（抽取前） 抗镜像（内插后）
    多相滤波 抽取后滤波 内插前滤波
    CIC 级顺序 先积分后梳状 先梳状后积分
    输入/出形式 实信号入，复基带出 复基带入，实信号出
